## Installation

Install my-project with npm

```bash
  cd Crypto Zombies
  cd client
  npm install 
  cd smart_contract
  npm install
  cd ..
  cd client
  npm run dev
```
    
## Tech Stack

**Client:** React, Redux, TailwindCSS

**Server:** Node
